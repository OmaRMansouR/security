# Flask SQL Injection
- Environment Setup
Run these 3 commands
```
cd flask_sql_injection
```
```
chmod 777 requirements.sh
```
```
./requirements.sh
```

- Running Application
```
flask run
```
or
```
python3 app.py
```

- Test SQL Injection
```
python3 perform_injection.py
```
